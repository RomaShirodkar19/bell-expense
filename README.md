# Bell Expense Tracker 🔔💰

A full-stack expense tracking application built with the MERN stack to help you manage your personal finances effectively.

## 🛠️ Tech Stack

**Frontend:**
- React.js with Vite
- JavaScript (ES6+)
- Tailwind CSS

**Backend:**
- Node.js
- Express.js
- RESTful API

**Database:**
- MongoDB

**Tools & Libraries:**
- npm for package management
- Git for version control
